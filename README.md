## Terraform

Virtualbox provider:

https://registry.terraform.io/providers/terra-farm/virtualbox/latest/docs/resources/vm

Inicializar vm:

```
terraform init
terraform plan
terraform apply
```

Obs.:

A imagem instalada do debian via vagrant não é o bullseye e sim buster.
